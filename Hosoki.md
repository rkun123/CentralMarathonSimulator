---

HTMLの基礎
```
<html>
    <head>
        <title>「Chromeでいう上のタブのとこに出てくるタイトル」</title>
        <script type="text/javascript">
        //この中にJavaScriptをかく
        alert("Hello,world!!");
        </script>
    </head>
    <body>
        <style>
            //ここにCSS書く
            p{
                color:red;
            }
        </style>
        <!--ここに表示するタグ+文字入れる-->
        
        <p>たとえばこんなの。</p>
    </body>
</html>
```
